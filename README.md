#### 项目简介

该项目是基于Python的web类库Django开发的在线招聘平台，支持岗位发布、浏览和搜索功能。开发环境是Django 3.2 + MySQL 5.7

主要功能：
- 岗位发布：支持发布全职、兼职、实习、外包等多种类型的岗位
- 岗位浏览：按职位类型筛选，查看岗位详情
- 薪资管理：支持薪资范围设置
- 联系方式：支持电话和邮箱联系
- 后台管理：支持岗位审核和管理

### 运行步骤

1. 安装依赖包，运行命令：pip install -r requirements.txt
2. 恢复数据库，运行命令: python manage.py migrate
3. 运行项目，运行命令: python manage.py runserver
